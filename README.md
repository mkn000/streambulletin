# Stream Bulletin
※日本語おかしくなら申し訳ないです。

This extension monitors and displays currently live streams from followed sources.  
このアドンはとあるサイトを監視しており、フォロー中チャンネルなどを配信中の場合で示しています。
## Supported sites / 利用サイト
* niconama (community/channel)
* whowatch
* TwitCasting
* Openrec
* Youtube
